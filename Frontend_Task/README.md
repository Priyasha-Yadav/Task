# Frontend Task Projects

This repository contains a collection of frontend projects built using HTML and CSS. Each project is organized in its own directory and demonstrates various UI components and layouts. These are simple static web pages that can be opened directly in a browser.

## Projects

### 1. Amazon
- **Description**: A basic clone of the Amazon homepage, featuring a navigation bar, search functionality, and product categories.
- **Files**: `amazon/index.html`, `amazon/style.css`
- **How to View**: Open `amazon/index.html` in your browser.

### 2. Calculator
- **Description**: A visual representation of a calculator interface with buttons for numbers and operations.
- **Files**: `Calculator/index.html`
- **How to View**: Open `Calculator/index.html` in your browser.

### 3. Chess
- **Description**: A static chessboard layout with pieces positioned for the start of a game, using Font Awesome icons.
- **Files**: `Chess/index.html`
- **How to View**: Open `Chess/index.html` in your browser.

### 4. Cricket
- **Description**: A scorecard display for the India vs Sri Lanka ICC World Cup 2011 match, showing batting details in a table format.
- **Files**: `Cricket/index.html`
- **How to View**: Open `Cricket/index.html` in your browser.

### 5. Flipkart
- **Description**: A clone of the Flipkart e-commerce homepage, including navigation, categories, hero section, and footer.
- **Files**: `flipkart/index.html`, `flipkart/style.css`
- **How to View**: Open `flipkart/index.html` in your browser.

### 6. Ludo
- **Description**: A visual Ludo board with colored paths, homes, and star markers.
- **Files**: `Ludo/index.html`
- **How to View**: Open `Ludo/index.html` in your browser.

### 7. Sudoku
- **Description**: A basic Sudoku grid layout (static, no interactivity).
- **Files**: `Sudoku/index.html`
- **How to View**: Open `Sudoku/index.html` in your browser.

### 8. SuperHeroes
- **Description**: A simple page showcasing superhero images.
- **Files**: `SuperHeroes/index.html`
- **How to View**: Open `SuperHeroes/index.html` in your browser.

### 9. Card
- **Description**: A card component layout.
- **Files**: `Card/index.html`, `Card/style.css`
- **How to View**: Open `Card/index.html` in your browser.

## Images
The root directory contains various image files (e.g., `amazon.png`, `kidz_world_1.png`, etc.) used across the projects.

## How to Run
- Clone or download the repository.
- Navigate to the desired project folder.
- Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox).
- No server setup is required as these are static HTML/CSS pages.

## Technologies Used
- HTML5
- CSS3
- Font Awesome (for icons in Chess and Ludo projects)
